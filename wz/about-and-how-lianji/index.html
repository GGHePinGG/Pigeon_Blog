首先感谢你来看这篇文章！我相信你对我的世界联机有一定的不了解，希望这篇文章可以帮到你~！
虽然这篇文章是教大家如何免费与朋友联机，但是还是希望大家入正哦！
正版用户不会联机的话直接登录正版然后看模块 #联机方式 哦！

账号注册与进入游戏

注册一个皮肤站账号

这个市面上有一大堆的外置皮肤站点
这里推荐两个：
Littleskin
Blessing Skin
这篇文章将用 LittleSkin 作为演示。

1.注册一个 LittleSkin 账户
注意：在注册前，你需要拥有一个邮箱。

2.找到“快速配置启动器”模块，并且保留这个页面，后面有用！

使用第三方启动器进入游戏

使用 HMCL 进入游戏

你需要下载HMCL 以启动游戏。
1.打开HMCL，在“账户”下点击头像。

2.HMCL是非常方便的，给你提供了LittleSkin的登录入口，但是如果你要使用其他的登录方式的话，请遵循下列步骤。

2-1.点击“添加外置服务器”。
2-2.输入“快速配置启动器”模块中的 Yggdrasil API 认证服务器地址.
2-3.这时旁边会出现一个新的登录方式，这样就添加成功了。

3.在你添加的登录方式中输入你注册时的邮箱和密码，这样就大功告成了！

使用 PCL2 进入游戏

你需要下载PCL2 以启动游戏。
1.打开PCL2，下载一个你要玩的游戏版本。
2.点击版本选择→你下载的新版本（或者你要编辑的新版本）→设置（齿轮图标）→设置 完成步骤后找到“服务器”模块。

3.点击“登录方式”，选择“第三方登录：Authlib-injector 或 LittleSkin”
4.PCL2同样给予了一键设置为LittleSkin的按钮，点击它就可以了！
但是如果你要使用其他的登录方式的话，请查看下图并遵循下列步骤。

5.回到主页，输入你注册时的邮箱和密码，这样就大功告成了！

使用 BakaXL 进入游戏

你需要下载BakaXL 以启动游戏。
1.打开BakaXL，进入本体设置。
2.点击“账户与档案”，下拉找到“Minecraft 用户档案”，点击“新增一个档案”。
3.将游戏启动模式改为“外置登录”
4.依照下图来配置启动器，完成后点击确定。

5.启动任一版本时使用外置登录档案，就大功告成了！

使用 FSM3 进入游戏

由于该启动器开发者学业极度繁忙，预计在今年OR明年OR后年推出FSM4 敬请期待吧！

怎么联机？

这是这篇文章的重点！重中之重！（敲黑板）

局域网联机

局域网联机分本地局域网跟虚拟局域网。本地局域网联机的朋友只需要打开游戏→多人游戏→找到朋友开启的局域网游戏就可以玩了！
如果你是主机，请进入游戏后点击“在局域网开放”，设置好游戏模式就可以等待你身边的朋友加入辣！

虚拟局域网联机

这个方法很适合异地（你和你的朋友不在一个地方但是想一起玩的）联机，当然异地联机方式不止这一种，接下来给大家介绍多种联机方式

Radmin Lan 联机

这是一段鸽了很久的教程！！！！！！！（咕咕咕）
你需要下载Radmin Lan 来进行联机！
下载太慢？ 国内镜像 提取码:RadM
1.打开 Radmin Lan，创建一个网络。
Tips：你需要打开Radmin Lan在线才可以进行游戏！

中间蓝色方框点击“创建网络”，创建一个网络。
2.网络名称建议设置简单点好记的，这样建设你与你朋友的输入网络名称的长度。
TIPS：密码也设置简单点好记的！！！！
3.让朋友在Radmin Lan 界面内点击加入网络，输入你提供的网络名称和密码，这样你俩就处于一个局域网内辣！
4.启动Minecraft，在游戏内点击在局域网开放，记住端口。
5.在你电脑名称下面的数字是IP地址，完整IP地址为IP地址+端口，例如：114.514.191.810:1145
6.让朋友通过局域网游戏加入服务器或者输入IP地址就可以一起玩辣！
